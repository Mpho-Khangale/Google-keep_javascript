# Google Keep Clone

## Overview

A responsive Google Keep Clone built using HTML, CSS and JavaScript.

## Features

- Create Notes
- Archive Notes
- Delete Notes
- Search Notes
- Dark Mode
- Local Storage
- Responsive Layout

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
